# Calculadora de Partidas Rankeadas

### Desafio do curso de Lógica de Programação – DIO.

## Sobre o Projeto

Este repositório contém a solução do desafio proposto pela DIO, onde criamos um programa em Python que calcula o saldo de vitórias de um jogador e determina seu nível conforme as regras definidas no exercício.

## O objetivo é praticar:

Variáveis

Entrada e saída de dados

Estruturas condicionais

Funções

Laços de repetição

## Regras do Desafio

O programa deve:

Receber a quantidade de vitórias

Receber a quantidade de derrotas

Calcular o saldo utilizando a fórmula:
saldo = vitórias - derrotas

Classificar o nível conforme a tabela:

| Vitórias | Nível    |
| -------- | -------- |
| < 10     | Ferro    |
| 11 – 20  | Bronze   |
| 21 – 50  | Prata    |
| 51 – 80  | Ouro     |
| 81 – 90  | Diamante |
| 91 – 100 | Lendário |

O programa deve exibir:

Sua diferença de vitórias e derrotas é 35. Seu rank é Ouro.

## Arquivo

O código final se encontra no arquivo calculadora_rank.py.
